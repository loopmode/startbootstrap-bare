This is a single page app template is based on the 'bare' theme from the startbootstrap project: https://github.com/IronSummitMedia/startbootstrap
I extended it with a require.js and Backbone setup, loading AMD modules asychronously without optmization.
A Backbone Router and application are already set up. Page contents are top-level Bootstrap containers within the body. They are hidden via CSS and made visible by the application object when the route changes.
